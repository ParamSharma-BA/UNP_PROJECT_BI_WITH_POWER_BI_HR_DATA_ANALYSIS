# UNP_PROJECT_BI_WITH_POWER_BI_HR_DATA_ANALYSIS

I did this project in collaboration with UNP, where the primary objective was to analyze workforce data and uncover actionable insights around employee attrition, engagement, and overall HR performance. The dashboard was built in Power BI and spans seven analytical pages — covering employee demographics, attrition risk, gender-based attrition patterns, job satisfaction and engagement levels, compensation and working conditions, career progression and experience trends, and performance alongside work-life balance metrics. The dataset included a range of HR variables such as age, department, job role, salary bands, tenure, education, and employee ratings, enabling a multi-dimensional view of the workforce. Key insights derived from the analysis include identification of high-risk attrition segments, departments with low satisfaction scores, and patterns linking compensation gaps or overtime to employee turnover. Interactive slicers and visual filters were used to make the report dynamic and drill-down friendly for HR stakeholders. This project strengthened my ability to translate raw HR data into strategic narratives using Power BI, with a focus on clean visual design and business relevance.
